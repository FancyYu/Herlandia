Herlandia - 宁静乌托邦岛屿模拟器

一个基于 Three.js 构建的交互式 3D 虚拟岛屿模拟器。在这里，你可以自由移动、建造建筑，并感受不同天气下的治愈氛围。

✨ 功能特性

交互式建造：通过右侧工具栏在岛屿上添加小屋、工坊、喷泉和花园。

自由移动：使用 ⬆️/⬇️/⬅️/➡️ 或方向键控制你的角色在岛屿上探索。

全天气系统：支持一键切换“晴朗”、“鲜花节”、“细雨”和“静谧夜”四种氛围环境。

粒子特效：精美的花瓣飘落（鲜花节）与发光粒子系统。

云端存储：集成 Firebase，支持保存和加载你的岛屿布局。

响应式设计：适配各种屏幕尺寸，支持拖拽移动建筑位置。

🛠️ 技术栈

渲染引擎: Three.js (R128)

样式框架: Tailwind CSS

后端服务: Firebase (Authentication & Firestore)

部署: GitHub Pages (推荐)

🚀 快速开始

克隆仓库:

git clone [https://github.com/你的用户名/herlandia-utopia.git](https://github.com/你的用户名/herlandia-utopia.git)


本地运行:

直接在浏览器中打开 index.html 即可。

或者使用 VS Code 的 Live Server 插件以获得更好的体验。

配置 Firebase:
为了使保存功能正常工作，你需要在 index.html 的 firebaseConfig 中填入你自己的 Firebase 项目配置。
